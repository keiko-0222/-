進捗管理一覧システム（Excel自動検索版）セットアップ手順
======================================================

■ 1. フォルダ構成
以下のファイルと、検索対象のExcelマクロブック（.xlsm）を
すべて「同じ1つのフォルダ」に入れてください。

  このフォルダ/
    ├ index.html
    ├ server.js
    ├ package.json
    ├ start.bat
    ├ README.txt（このファイル）
    ├ Yahoo売上帳.xlsm          ← モール別のマクロブック
    ├ オフィシャル売上帳.xlsm
    ├ アマゾン売上帳.xlsm
    ├ ギフトモール売上帳.xlsm
    ├ うちのこいちばん売上帳.xlsm
    ├ ふるさと納税売上帳.xlsm
    ├ 社内売上帳.xlsm
    └ 4you売上帳.xlsm

※ ポイント：Excelファイルの「ファイル名」に、検索画面のラジオボタンと
  同じ文字列（yahoo / オフィシャル / アマゾン / ギフトモール /
  うちのこいちばん / ふるさと納税 / 社内 / 4you）が含まれている必要があります。
  含まれていれば、ファイル名の前後に他の文字（日付など）が付いていても
  問題なく検索されます。
  例）「2025年7月_オフィシャル売上帳.xlsm」→「オフィシャル」を選ぶとヒット

※ 同じモール名を含むファイルが複数ある場合（例：年度ごとに分かれている等）は、
  該当するファイルすべてをまとめて検索します。

■ 2. 事前準備（最初の1回だけ）
Node.js がパソコンに入っていない場合は、以下からインストールしてください。
  https://nodejs.org/ （「LTS」版でOKです）

■ 3. 起動方法
「start.bat」をダブルクリックしてください。
- 初回のみ、自動で必要なファイルのインストール（npm install）が始まります。
  インターネット接続が必要です。数分かかることがあります。
- インストールが終わると自動でブラウザが開き、画面が表示されます。
- 次回からはダブルクリックするだけで、すぐに画面が開きます。
- 黒い画面（コマンドプロンプト）は、使っている間は閉じずにそのままにしておいてください。
  閉じるとシステムが使えなくなります。

■ 4. 使い方
1) 画面上部の「対象モール」で検索したいモールを選びます。
2) 「検索条件」で「注文日で検索」か「お客様番号で検索」のどちらかを選びます。
   - 注文日で検索：開始日と終了日を指定（同じ日付にすれば1日だけの検索）
   - お客様番号で検索：開始番号と終了番号を指定（例：123～200）
     数字だけの番号（2600001など）にも、アルファベットで始まる番号
     （A045など）にも対応しています。
3) 「🔍 検索してこのシステムに反映」ボタンを押すと、該当するExcelの
   集計表シートからデータを取得し、一覧表に反映されます。
4) 一覧表に反映された後の操作（ステータス変更、CSV/PDF出力など）は
   これまでと同じです。

■ 5. 注意事項
- 検索のたびに、その時点の一覧表の内容は新しい検索結果に置き換わります
  （ステータスなど、検索結果に反映していない手動の変更は失われます。
  　CSV/PDF出力など必要な操作は、次の検索を行う前に済ませてください）。
- 対象のExcelファイルをExcelで開いたまま検索すると、Excel側で保存していない
  最新の変更は反映されません。検索前に保存（またはExcelを閉じる）ことを
  おすすめします。
- 「集計表」シートの列構成が変わった場合は、server.js の中ほどにある
  「COL」という設定部分の数字（列の位置）を実際の列に合わせて
  書き換えてください（A列=0, B列=1 ... という並びです）。

■ 6. うまく動かないとき
- ボタンを押しても反応がない／エラーが出る場合は、start.bat を実行した
  黒い画面が開いたままになっているか確認してください。
- 「Excelファイルが見つかりません」と出る場合は、ファイル名にモール名の
  文字列が含まれているか、正しいフォルダに置かれているか確認してください。
- 列がずれて違う項目が表示される場合は、上記5の「COL」設定をご確認ください。
